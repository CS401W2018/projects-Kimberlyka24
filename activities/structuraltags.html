Structural Tags (HEADER, NAV, MAIN, FOOTER)

What goes in the header?

The header tag is used for the introductory part of the website

What goes in the Nav? Can the Nav go in the header? Does it have to go in the header? Does the Nav contain links to another website?

The nav tag is basically navigation. It contains links that will send you to different parts of the website (or even external links). It can be inside the header tag or even outside the header tag.

What goes in Main?

The main tag has all the main content of the web page, such as articles, videos, paragraphs, etc.

What goes in the Footer?

The footer tag usually contains the ending part of the web page, such as copyright, contact info, social media links, etc.

HTML Validation

HTML Validation is a tool used to check if the HTML code created complies with the standard rules of the Web. Most commonly used validator is the online tool called W3C Markup Validation Service.
