What is the difference between absolute link and a relative link?

The difference between absolute link and relative link is that absolute links contains the whole web address and usually are external links while relative links are links that are within the same directory or same site the current page is located. So basically, we user absolute links for external links while we use relative links for links within the same site.

Three Rules with relative links

  • A single dot (.) means the link is in the current folder
  • A double dot (..) means it is in the parent folder
  • A slash (/) means it starts at the root

Do link always need to be inside a NAV tag?

It is not required for links to be inside the nav tag. Nav tag's purpose is to group links for navigation purposes, but not necessary to put all of the links there.
