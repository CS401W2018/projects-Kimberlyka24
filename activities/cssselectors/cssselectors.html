CSS Selectors

List out the different ways to select an item.

The different ways to select an item are universal selctor (*), type selector (p, h1, etc.), class selector, id selector (#), attribute selector, descendant selector, child selector, and many other more. Depending on the use-case, you can use some or all of them in a single css file.

What is the difference between an ID and a Class?

Their are two main difference between ID and Class. The first is the syntax, ID uses asterisk (#) while class uses dot (.). An ID can only be used once in a page while class can be multiple.

What is the difference between a SPAN tag and a DIV tag?

They are both used as a container. The difference is that DIV is used to group large sections of the HTML while SPAN is used for small portions of text in an HTML.

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

While HEADER, MAIN, and FOOTER are used to structure different parts of the webpage, DIV is flexible and can be used anywhere to group a big portion of an HTML.

What are the different ways to select a color?

The different ways to select a color in an HTML are using named colors (such as red, blue, green, etc.), hexadecimal (example: #ff5733), rgb, rgba, and many other more. They have different advantages like named colors are easy to code while hexadecimals are used to get more specific color space.
