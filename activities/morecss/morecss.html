Engagement - More CSS

  1. What is the difference between block tags and inline tags?

    The main difference between block and inline tags is that a block tag is written in a new line while an inline tag is written inside an existing line. Block tags are usually used in defining large parts of a webpage while inline tags are mostly used for styling parts of an element.

  2. What is the margin, border, and padding?

    Margin, border, and padding are used to control the layout of the elements in a webpage. Margin is space outside of the element, border is for wrapping around the element, and padding is space inside the element.

  3. What are units of measurement you can use in CSS?

    The units of measurement in CSS are categorized into two parts: Absolute and Relative Units. Absolute units (px, pt, cm, mm, in) are fixed units that does not change. Relative Units (em, rem, %, vw, vh, vmin, vmax) are units that scale depending on the size of the webpage.

  4. When would you use a figure tag?

    You can use <figure> if you want to encapsulate an image or a video and add a caption.
