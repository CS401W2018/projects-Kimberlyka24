  3. What is the main difference between CSS Grid and CSS Flex?

    The main difference between the two is that grids are used for more complex arrangements of elements in an html while flex are mostly used for simpler layouts. The best use-case would be to use the grid for the main layout, then yuse flex for the internal elements.
