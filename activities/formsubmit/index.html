  • What is JavaScript? How did it get its name?

    JavaScript is a programming language mainly used in web browsers. It was formerly called Mocha in its initial phase, but it was later changed by Netscape to LiveScript in 1995. At the time, they were working with Sun Microsystems, the creator of Java. Java was popular, so they changed LiveScript to JavaScript.

  • What are the two ways you can run JavaScript?

    JavaScript can be run client-side and server-side.

  • What was the most popular front-end js library?

    The most popular front-end js library was jQuery.

  • What is the most popular front-end js library today?

    Today, React is the most popular front-end js library.
