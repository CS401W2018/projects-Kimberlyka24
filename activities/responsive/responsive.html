  1. Why do we need a viewport meta tag?

    Viewport meta tag enables a page to become responsive, making it accessible to all devices of different sizes.

  2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

    The one that appears last takes priority, so in this example, it would be flex-direction: row.

  3. How do you change the CSS for a web page displayed on a phone?

    You can use media queries to change the size or how the elements are shown in a smaller screen like a phone.
