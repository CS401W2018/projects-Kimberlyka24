Engagement - Page for Print

  1. Why is the viewport meta tag important?

    Viewport meta tag is important because it controls how a webpage is viewed on different screen sizes or devices. By setting it up correctly, a website will accomodate different kinds of devices.

  2. How can you adjust the CSS for printing?

    To adjust CSS for printing, you can use the following: use media queries, create separate stylesheet, adjust layout and elements, and control page breaks.

  3. When would you want to adjust the CSS for printing?

    The best time to adjust CSS for printing is for reporting/documenting, articles/blogs, forms, etc.
