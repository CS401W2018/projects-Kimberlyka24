DNS and SSL

  • What is DNS?

    DNS or Domain Name System is basically the system used to translate domain name of a website to IP addresses.

  • How do you get a domain name?

    Basically, you can buy a domain name that you like, if it is available. Sites like GoDaddy, Google Domains, Namecheap, etc. offer services where you can buy a domain name for your site.

  • What is SSL?

    SSL or Secure Sockets Layer is the system that protects the data that is being transferred between the website and the user. It encrypts and authenticates every data that is being transferred.

  • How do you get a security certificate?

    First you pick a CA or a certificate authority organization. Second, you choose which type of SSL certificate you want. Next, generate a Certificate Signing Request. Fourth, order the SSL certificate. Then lastly, Install the SSL certificate.
