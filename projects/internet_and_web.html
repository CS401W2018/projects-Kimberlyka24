Internet

The internet is the largest computer network created to exchange data throughout the world such as watching videos, sending e-mails, downloading games, etc.

Web

The web is a part of the internet that users access to navigate the internet, basically the system we use to interact with the internet.

How are the Internet and Web different

The difference between the two is that the Web is part of the Internet. The web is accessed through the internet.

ASCII and Unicode

ASCII and Unicode are character encoding standards used by the computers to encode alphanumeric keys to machine language, basically letters like "A, B, and C" can be encode to binary numbers (0 and 1). The difference is that ASCII uses 7 bits (only up to 12 characters) while Unicode can encode more characters.

What is UTF-8

UTF-8 is an encoding form of Unicode. It uses 8 bits, contrary to ASCII that only uses 7 bits. Also, all ASCII-based systems are compatible with UTF-8.

URL

URL is the address used to visit a specific website.

Domain Name

A structure of the URL that specifies the name of a website that you want to visit.

HTTP

HTTP means Hypertext Transfer Protocol. This protocol handles the transmission of webpages to and from the clients and the servers.

HTTPS

HTTPS is similar to HTTP but more secure. It uses encryption to make each visit secure.

IP Address

is the numerical address given to networks connected to the internet. each network has its own unique ip address.

DNS

DNS means Domain Name System. Basically what it does is translates domain names into IP addresses. So the computers read the IP address while a user types the domain name.

W3C Recommendations

W3C Recommendations is the standard that the internet uses to make it stable, organized, and future-proof.

XML, XHTML, HTML, HTML5

XML (Extensible Markup Language) - is a markup language used for storing and transporting information.
XHTML (Extensible Hypertext Markup Language) - a markup language that combines XML and HTML.
HTML (Hypertext Markup Language) - the markup language commonly used by websites, basically the standard or basis that every website use.
HTML5 - the latest version of HTML. It improves stability, built in plugins to remove dependancy from plugins like flash.
